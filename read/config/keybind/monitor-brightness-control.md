# 螢幕亮度控制


## 螢幕亮度控制

* [設定片段](https://github.com/samwhelp/note-about-labwc/blob/gh-pages/_demo/config/labwc-config/main/rc.xml#L251)


| 按鍵組合          | 功能             | 執行指令                                    |
| ----------------- | ---------------- | ------------------------------------------- |
| `Alt + Shift + [` | 減少螢幕亮度         | `brightnessctl set -10%` |
| `Alt + Shift + ]` | 增加螢幕亮度         | `brightnessctl set +10%` |
